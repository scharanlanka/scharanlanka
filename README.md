## Featured Projects

### [Agentic RAG with Retrieval Inspector](https://github.com/scharanlanka)
Agentic RAG pipeline using LangChain, LangGraph, Azure OpenAI, and ChromaDB. Includes a retrieval inspector to analyze embedding quality, chunking strategies, and retrieval relevance — improving grounding and reducing hallucinations.

Self-improving research agent with a Planner → Researcher → Summarizer → Critic workflow. Retrieves from Tavily web search and arXiv, generates cited reports, and uses ChromaDB-backed long-term memory for iterative knowledge refinement.
